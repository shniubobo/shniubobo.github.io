Hello, world!

打算建个博客，目前只做了一些基本的配置，看上去很简陋，各种方面都还有许多需要完善之处：

  • 网站没有图标
  • 缺少指向源码以及我的 Github 页面的链接
  • 这个主题的标签和分类页面无法禁用，始终会在页面上显示相应链接
  • ……

始终没有找到一个满意的主题，打算以后自己写一个，顺便可以当做学习 HTML 和 CSS 时的练习吧。

可能会在博客发各种东西，不局限于任何一个领域，主要还是看心情吧。

花了大半个国庆假期，终于把主题基本做好了。上张图对比下：

说是基本做好是因为现在还有很多功能不支持，例如图片没有做 lazyload（这个可能需要自己写个插件自动生成缩略图并为 <img> 标签加上 data-src 属性）、没有目录（这个应该有现成的插件可以用）等，同时还有一些地方看着有些不顺眼（例如下面的许可信息）。放假一结束就有好几个 DDL，所以打算先把 DDL 赶完了再来做剩下的主题。主题完善后还会考虑做一些小修改再开源（例如右边这几个社交图标不方便用在通用的主题里）。

做主题的整个过程里踩了很多坑，也会在赶完 DDL 后写一篇文章讲一下。
